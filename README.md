### Como Instalar no Celular (PWA)

#### 📱 Android:
1. Abra o link no **Chrome** do Android
2. Toque no menu (3 pontos) do navegador
3. Selecione **"Adicionar à tela inicial"** ou **"Instalar app"**
4. Confirme a instalação
5. O app aparecerá na tela inicial como um app nativo!

#### 🍎 iPhone/iPad:
1. Abra o link no **Safari** do iOS
2. Toque no botão **"Compartilhar"** (quadrado com seta)
3. Selecione **"Adicionar à Tela de Início"**
4. Confirme a instalação
5. O app aparecerá na tela inicial!

## 📋 Sobre o Projeto

O **Marido de Aluguel Carioca** é uma plataforma digital inovadora que conecta clientes a prestadores de serviços residenciais qualificados no Rio de Janeiro. Desenvolvido como um Progressive Web App (PWA), o aplicativo oferece uma experiência nativa tanto em dispositivos móveis quanto em desktops, facilitando a contratação de serviços domésticos de forma rápida, segura e eficiente.

### 🎯 Objetivo

Resolver a dificuldade que muitas pessoas enfrentam para encontrar profissionais confiáveis para pequenos reparos e serviços domésticos, criando uma ponte digital entre clientes e prestadores de serviços especializados.

- **Instalável**: Pode ser instalado na tela inicial como um app nativo

## 🛠️ Como Executar Localmente

### Pré-requisitos:
- Node.js 18+ 
- npm ou yarn

### Instalação:

```bash
# Clone o repositório
git clone https://github.com/eusoumanoelnetto/marido-de-aluguel-carioca.git

# Entre no diretório
cd marido-de-aluguel-carioca

# Instale as dependências
npm install

# Execute em modo de desenvolvimento
npm run dev

# Para build de produção
npm run build
```

### Scripts Disponíveis:

```bash
npm run dev          # Servidor de desenvolvimento
npm run build        # Build para produção
npm run preview      # Preview do build de produção
npm run lint         # Verificação de código
```

## 🔄 Fluxo de Uso

1. **Entrada**: Cliente acessa o aplicativo via web ou PWA instalado
2. **Autenticação**: Login simples (simulado para demonstração)
3. **Seleção de Serviço**: Escolha da categoria principal
4. **Sub-serviço**: Seleção do tipo específico de serviço
5. **Solicitação**: Preenchimento de formulário detalhado
6. **Orçamentos**: Recebimento de propostas de prestadores
7. **Contratação**: Escolha do prestador e finalização
8. **Acompanhamento**: Chat e avaliação do serviço
